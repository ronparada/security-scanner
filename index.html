Advanced Security Scanner

Note: This is a static demo version of the security scanner. For the full functionality, please run the application locally using the instructions in the README.

Features

  • SSL Certificate Analysis
  • Security Headers Check
  • Port Scanning
  • Subdomain Enumeration
  • Directory Scanning
  • Technology Stack Detection
  • Email Security (SPF, DMARC, DKIM)

How to Use

  1. Clone the repository: 
    git clone https://github.com/ronparada/security-scanner.git
cd security-scanner
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Run the application: 
    python app.py
  4. Open your browser and navigate to http://localhost:5000

Security Note

This tool is for educational and testing purposes only. Always obtain proper authorization before scanning any domain.
